# Reguli Stricte — md-buildings

## Scop
Icoane pentru secții legate de locații fizice, spitale, clinici, domicilii.

## Domeniu
**Clădiri, case, construcții**

## Format obligatoriu
- Toate fișierele: `.webp`
- Nomenclatură: `md-ico-{descriptor}.webp`
- Prefix unic: `md-ico-`

## Reguli de actualizare
1. Orice adăugare **trebuie** să respecte nomenclatura `md-ico-{descriptor}.webp`
2. Nu se acceptă formate PNG/SVG/JPEG
3. Actualizați acest registru la fiecare modificare
4. Ștergerea unui fișier necesită verificarea că nu este referit în codul sursă
